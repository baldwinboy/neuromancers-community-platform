# Dashboard Overview

The Wagtail admin dashboard is your central hub for managing the Neuromancers platform.

## Features

From the dashboard you can:

- **View recent activity** - See the latest changes made to the site
- **Access quick links** - Jump to frequently used areas
- **Monitor site health** - Check for any issues or warnings

## Platform Guide Button

A prominent "Platform Guide" button appears on the main dashboard page, providing quick access to this documentation. You can also find it in the help menu in the sidebar.

## Navigation

Use the sidebar menu to navigate between different sections:

- **Pages** - Manage site content
- **Images** - Upload and manage media
- **Documents** - Manage downloadable files
- **Sessions** - View and manage peer/group sessions
- **Users** - User administration
- **Settings** - Site configuration
