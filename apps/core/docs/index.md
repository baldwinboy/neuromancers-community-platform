# Neuromancers Platform Guide

Welcome to the Neuromancers Community Platform administration guide. This documentation covers all aspects of managing the platform.

Use the navigation menu on the left to explore different sections.

---

## Topics Covered

- **Dashboard Overview** - Understanding the admin dashboard and key metrics
- **Managing Sessions** - Create, edit, and manage peer and group sessions  
- **User Management** - Handle user accounts, roles, and permissions
- **Web Design Settings** - Customize fonts, colors, branding, and styling
- **Email Templates** - Configure notification emails and templates
- **Content Management** - Manage pages, blogs, and media content

## Quick Start

The Wagtail admin dashboard is your central hub for managing the Neuromancers platform. From here you can:

- **View recent activity** - See the latest changes made to the site
- **Access quick links** - Jump to frequently used areas
- **Monitor site health** - Check for any issues or warnings
