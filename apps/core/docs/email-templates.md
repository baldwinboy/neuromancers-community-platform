# Email Templates

Access **Settings > Email Templates** to customize notification emails.

Each email type can be customized with greeting text, body content, and closing notes.

## Customizable Fields

For each email type:

- **Greeting** - Opening text shown at the top
- **Body** - Main message content
- **Note** - Additional information (where applicable)

All fields support rich text formatting.

## Email Types

### Account Emails

#### Account Created

Sent when a new user registers.

- Welcomes the user
- Encourages profile setup
- Links to getting started resources

#### Account Closed

Sent when a user closes their account.

- Confirms account closure
- Provides information about data retention
- Offers re-registration option

### Group Status Emails

#### Group Upgrade (to Peer)

Sent when a user is upgraded to Peer status.

- Congratulates the user
- Explains new capabilities
- Links to session creation guide

#### Group Downgrade

Sent when a user's role changes.

- Notifies the user of the change
- Explains available features
- Provides support contact

### Session Emails

#### Session Published

Sent to the host when their session is published.

- Confirms publication
- Explains visibility to seekers
- Links to session management

#### Session Requested

Sent to the host when someone requests their session.

- Notifies of new request
- Includes requester details
- Links to approve/decline

#### Session Approved

Sent to the seeker when their request is approved.

- Confirms the booking
- Includes session details
- Provides joining instructions

### Payment Emails

#### Payment Made

Sent to the seeker after successful payment.

- Confirms payment
- Includes receipt information
- Links to session details

#### Payment Received

Sent to the host when they receive payment.

- Notifies of payment
- Includes payment details
- Links to Stripe dashboard

### Refund Emails

#### Refund Requested

Sent to the host when a refund is requested.

- Notifies of refund request
- Includes reason (if provided)
- Links to approve/decline

#### Refund Approved

Sent to the seeker when refund is approved.

- Confirms refund
- Includes processing timeline
- Provides support contact
