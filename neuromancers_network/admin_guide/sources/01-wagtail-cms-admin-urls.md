# Wagtail CMS Admin URLs

This page covers only one topic: every major Wagtail admin URL that starts with `/cms/`.

Use this page when you want to know where to go in the CMS before changing anything.

| URL | What it does | What you can change there |
| --- | --- | --- |
| `/cms/` | Wagtail dashboard | See your recent edits, pages waiting for moderation, and shortcuts to common tasks. |
| `/cms/pages/` | Page tree | Create pages, edit existing pages, move pages, publish pages, unpublish pages, and delete pages. |
| `/cms/images/` | Image library | Upload logos, photos, and illustrations used across the site. |
| `/cms/documents/` | Document library | Upload PDFs, spreadsheets, and downloadable files. |
| `/cms/settings/` | Site settings | Change branding, email settings, auth-page styling, terminology, footer, navbar, security, and API keys. |
| `/cms/snippets/` | Reusable content | Manage reusable pieces of content if snippets are registered in the project. |
| `/cms/forms/` | Form submissions | Read replies sent through Contact Form pages. |
| `/cms/redirects/` | Redirects | Create redirects from old URLs to new URLs. |
| `/cms/reports/` | Reports | Review locked pages, old pages, and other site-level reports. |
| `/cms/searchpicks/` | Search promotion | Promote specific results for selected search terms. |

## Which URL should you start with?

- If you want to change **page content**, start at `/cms/pages/`.
- If you want to change **site-wide design or settings**, start at `/cms/settings/`.
- If you want to upload **images**, start at `/cms/images/`.
- If you want to upload **PDFs or other files**, start at `/cms/documents/`.

## Accessibility notes

- The Wagtail admin is keyboard-navigable.
- If you use a screen reader, page headings and form labels are usually the fastest landmarks.
- If the page tree feels overwhelming, use the search field near the top of the Pages area to find a page by title.
